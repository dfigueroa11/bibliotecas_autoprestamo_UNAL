# Autoprestamo de libros
# Bibliotecas Universidad Nacional de Colombia
Con el objetivo de construir un campus inteligente dentro de la Universidad Nacional de Colombia-Sede Bogotá se busca suplir algunos de los servicios que son ofrecidos a la comunidad universitaria, dentro de los que se encuentra el autoprestamo de libros dentro de la Red de Bibliotecas dentro del campus universitario.
El proceso de autoprestamo de libros en las bibliotecas contempla un antes, durante y despues por parte del usuario
